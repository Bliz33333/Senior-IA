Detailed Description

The file that contains how many and the names of the Literatures.

This file contains the number of IB literatures in the first line, the number of AP literatures in the seconds, then each IB literature title in its own line, and finally all the AP literature titles each in their own line. Each title corresponds to the name two files (title.txt and titlePD.txt) that contain info for the literature. Formatted like below: 

number ib literatures
number ap literatures
1st ib lit title
2nd ib lit title
.
.
.
last ib lit title
1st ap lit title
2nd ap lit title
.
.
.
last ap lit title
